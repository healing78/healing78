벙주가 해야 할 일

벙주는 아래 내용을 잘 이행바랍니다.

1. 친구들을 벙대기실로 안내하기
(간혹 벙대기실 입장 안하는 친구가 있습니다)

2. 새친구가 있을 경우 벙진행이나 힐링방에 대해서 잘 소개부탁합니다.

3. 차수별로 벙장소 알려주기
(장소 이동시에도 벙대기실에 알려주세요)

4. 차수별로 벙비 정산을 투명하게 하기
(골든벨 등도 확인 필수입니다)

5. 벙중에 성추행 폭력 싸움 등 불미스러운 일들이 발생하지 않도록 잘 부탁합니다.

6. 벙참석자 확인 가능한 사진 첨부하기
(참석했는지 판별은 오로지 사진뿐)

7. 지인 초대시 친구들에게 동의를 구하기

8. 벙마감후 친구들 안전 귀가를 확인 하기
(집도착하였다고 안전귀가톡 달도록 안내할 것)

9. 마감 댓글 작성을 하기
